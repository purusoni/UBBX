# UBBX — UB Book Exchange

UB Book Exchange — CSE 426 blockchain project.

Peer-to-peer textbook listings on Ethereum (Sepolia), priced in **Yoda**.  
**Smart contract:** [`contracts/UBBX.sol`](contracts/UBBX.sol)  
**Web app (GitHub Pages root):** [`UBBXDapp/src/`](UBBXDapp/src/) — set `app.js` contract addresses after deploy.

Deploy the static files under `UBBXDapp/src/` to Pages, or serve them with any static host. See [`DEPLOYMENT_AND_AI.md`](DEPLOYMENT_AND_AI.md) for Remix, metadata JSON, and AI disclosure notes.
