# UBBX — UB Book Exchange

UB Book Exchange — CSE 426 blockchain project.

Peer-to-peer textbook listings on Ethereum (Sepolia), priced in **Yoda**.  
**Smart contract:** [`contracts/UBBX.sol`](contracts/UBBX.sol)  
**Web app** lives at the **repo root** (`index.html`, [`app.js`](app.js), [`metadata/`](metadata/)).  
**GitHub Pages:** Settings → Pages → branch **`main`**, folder **`/ (root)`**. Edit [`app.js`](app.js) for contract addresses.

Local test: `python3 -m http.server 8080` in the repo root. See [`DEPLOYMENT_AND_AI.md`](DEPLOYMENT_AND_AI.md) for Remix and metadata.
