# UBBX — UB Book Exchange

UB Book Exchange — CSE 426 blockchain project.

Peer-to-peer textbook listings on Ethereum (Sepolia), priced in **Yoda**.  
**Smart contract:** [`contracts/UBBX.sol`](contracts/UBBX.sol)  
**Web app (GitHub Pages):** [`docs/`](docs/) — in repo **Settings → Pages**, set source to **`main`** and folder **`/docs`**. Edit [`docs/app.js`](docs/app.js) for contract addresses.

For local testing: `cd docs && python3 -m http.server 8080`. See [`DEPLOYMENT_AND_AI.md`](DEPLOYMENT_AND_AI.md) for Remix and metadata.
